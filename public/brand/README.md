# Brand — logo & foto kepala sekolah

Disajikan di URL **`/brand/...`** dari root `public/brand/`.

| Berkas | URL | Dipakai di |
|--------|-----|------------|
| `logo.png` | `/brand/logo.png` | Navbar, footer, login, loading (`BRAND_LOGO_SRC`) |
| `kepala-sekolah.png` | `/brand/kepala-sekolah.png` | Sambutan kepala sekolah beranda/profil (`BRAND_KEPALA_FOTO_SRC`) |

Ganti berkas **in-place** (nama tetap) untuk rebrand. Konstanta: `src/lib/branding.ts`.

Media konten halaman (hero, fasilitas, ekskul): lihat [`../media/README.md`](../media/README.md).
